PD:manera de instalacion:

Descomprimes el RAR y vacias todo el contenido en una carpeta ,subes la carpeta al FTP y vas a www.tudominio.com/foro(o como se llama la carpeta donde pusiste el foro)/install.php

seria:

www.tudominio.com/foro/install.php

hay tienes que poner :

el servidor de la base de datos, el nombre de tu comunidad,el nombre de la base de datos,el user de la base de datos y la pass despues de esoo te sale una pag. que te hace crear el nombre de user y la pass del admin principal del foro y listo foro instalado.
