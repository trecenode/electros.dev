[b]Aviso:[/b] Gracias a [b]kzale[/b] que me dió información detallada de los errores que tenía este script, lo he actualizado y ya debe funcionar correctamente.

Con este script todos tus usuarios tendrán un perfil de usuario que podrán editar en cualquier momento, también podrás mostrar una lista de todo los usuarios registrados de tu web en el cuál podrás ver el perfil de cada uno de ellos.

El script se compone de 3 archivos:

[b]uperfil.php[/b]
[b]ueditar.php[/b]
[b]usuarios.php[/b]

El funcionamiento de los archivos es el siguiente:

[b]uperfil.php[/b]
En esta página el usuario podrá editar su perfil.

[b]ueditar.php[/b]
Esta página procesará el formulario y actualizará los datos en la base de datos.

[b]usuarios.php[/b]
Esta página mostrará todos los usuarios registrados de tu web, y también podrás ver el perfil de cada usuario.

Las páginas de uperfil.php y usuarios.php puedes enlazarlas directamente o meterlas en otra página, la página de ueditar.php debe estar totalmente sóla o te dará errores.

Primero hay que actualizar la tabla de los usuarios y agregar los campos que hacen falta, para esto mete el siguiente código en el phpMyAdmin:

[codigo]alter table usuarios
add pais varchar(20) not null,
add edad tinyint(2) unsigned not null,
add sexo enum('0','1') not null,
add descripcion tinytext not null
after email[/codigo]

[b]Explicación de su uso[/b]

El archivo uperfil.php puedes ponerlo en el menú de usuario registrado, para que de esta forma el usuario tenga acceso a su perfil y lo pueda editar cuando el lo desee.

El archivo ueditar.php debe estar totalmente sólo, si usas un sistema de enlaces del tipo index.php?id=pagina no metas la página de ueditar.php o te dará errores de Cannot add headers.

La página usuarios.php puedes ponerla en cualquier parte de tu web ya que sirve para mostrar a todos los usuarios registrados de tu web y para poder ver su perfil.
