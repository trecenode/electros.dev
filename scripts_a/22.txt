[codigo]create table mensajes (
id smallint(5) unsigned not null auto_increment,
nuevo tinyint(1) unsigned not null,
fecha int(10) unsigned not null,
remitente varchar(20) not null,
destinatario varchar(20) not null,
mensaje tinytext not null,
primary key(id),
index (destinatario)
)[/codigo]

[b]Explicación de su uso[/b]

Sólo debes crear un enlace hacia el archivo mensajes.php para que desde ahí puedan se puedan enviar, responder y borrar mensajes.
