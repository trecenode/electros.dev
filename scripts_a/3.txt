Este sistema te permitirá poder enviar noticias a tu web, para que de esta forma no tengas la necesidad de estar actualizando constantemente tu página, también tendrás la opción de que tus visitantes puedan escribir comentarios, y así, puedas saber las distintas opiniones sobre la noticia.

El sistema se compone de 3 archivos:

[b]noticiasenviar.php[/b]
[b]noticias.php[/b]
[b]noticiascom.php[/b]

El funcionamiento de los archivos es el siguiente:

[b]noticiasenviar.php[/b]
En esta página es donde se envían las noticias a tu web.

[b]noticias.php[/b]
Aquí se mostrarán las noticias que se encuentren almacenadas en la base de datos.

[b]noticiascom.php[/b]
Esta página guardará los comentarios en la base de datos y regresará a la página con la noticia.

Primero debes crear las tablas noticias y noticiascom en la base de datos, si usas el phpMyAdmin debes copiar el siguiente código y pegarlo en donde dice SQL.

[codigo]create TABLE noticias (
id tinyint(3) unsigned not null auto_increment,
fecha int(10) unsigned not null,
usuario varchar(20) not null,
titulo varchar(100) not null,
noticia tinytext not null,
noticiaext text not null,
primary key (id)
) ;
create TABLE noticiascom (
id smallint(5) unsigned not null auto_increment,
noticia tinyint(3) unsigned not null,
fecha int(10) unsigned not null,
usuario varchar(20) not null,
comentario tinytext not null,
primary key (id),
index (noticia)
)[/codigo]

[b]Como integrar el sistema de noticias al sistema de registro de usuarios[/b]

Para aquellos que les interese deben realizar los siguientes pasos:

[b]1. Modificando el archivo noticiasenviar.php[/b]

[b]-[/b] Primero abran la página de noticiasenviar.php en un editor de texto o de html puro.
[b]-[/b] Luego agreguen hasta arriba el siguiente código:

[codigo]&lt;?
include(&quot;ulogin.php&quot;) ;
?&gt;[/codigo]
[b]-[/b] Después deben modificar la línea 10 y agregar en la variable $usuario el valor $_COOKIE[&quot;unick&quot;] de esta forma:

[codigo]$fecha = time() ;
$usuario = $_COOKIE[&quot;unick&quot;] ;
$comentario = quitar($comentario) ;[/codigo]
[b]-[/b] Por último sólo eliminen las líneas 20 y 21 que son para que el usuario agregue su nick.

[codigo]Usuario:&lt;br&gt;
&lt;input type=&quot;text&quot; name=&quot;usuario&quot; maxlength=&quot;20&quot;&gt;&lt;br&gt;[/codigo]

[b]2. Modificando el archivo noticias.php[/b]

[b]-[/b] Abran la página de noticias.php.
[b]-[/b] Eliminen las líneas 72 y 73 que es para que el usuario pueda escribir su nick.

[codigo]Usuario:&lt;br&gt;
&lt;input type=text name=usuario maxlength=20&gt;&lt;br&gt;[/codigo]

[b]3. Modificando el archivo noticiascom.php[/b]

[b]-[/b] Vuelvan a crear el archivo noticiascom.php con el siguiente código:

[codigo]&lt;?
