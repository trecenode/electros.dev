Es un Script muy sencillo, como el que podeis encontrar en la Web de PHPMySql, pero éste incorpora las bases de datos.

Explico sencillamente como lo hace.

En la primera parte, envías el formulario, en el que coge tu nick de la cookie, etc..

En la segunda parte borra todo el contenido de la tabla de mensajes publicos y despues envía los datos introducidos en el formulario.

Es muy sencillo, pero puede resultar útil para poner mensajes en marquesinas, o de titulares, etc..etc..

Bueno, no me enrollo más, aki está el código:


[b] Introducir en una tabla de mysql [/b]

[codigo]

CREATE TABLE `mpublico_mensajes` (
  `mensaje` varchar(255) NOT NULL default '',
  `nick` varchar(40) NOT NULL default '',
  `fecha` varchar(20) NOT NULL default '',
  `ip` varchar(20) NOT NULL default '',
  `msm` varchar(12) NOT NULL default ''
) TYPE=MyISAM COMMENT='Tabla para el sistema de mensajes publicos';

[/codigo]
