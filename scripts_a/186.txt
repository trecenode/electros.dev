ElectrosNuke es un script creado por phpmysql, es la recopilación de los scripts de Electros y phpmysql para crear un portal con todo lo necesario para un webmaster que busca una solución fácil y rápida.

De todas las versiones echas de electrosnuke que salgan

Y el diseño no es el definitivo es un cualquiera que he elegido al azar y luego cambiara.

El script funciona de la seguiente manera, tiene el eforo 2.1 y el registro de usuario de electros : noticias,minichat,mensajes,usuarios online....
Para instalarlo solo debes modificar el archivo [b]config.php[/b] y abrir el archivo [b]instalar.php[/b] y se instalara solo ( no requiere conocimientos avanzados para hacer este proceso)
