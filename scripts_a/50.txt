Con este script podrás hacer que tus usuarios puedan recuperar su contraseña, para esto el usuario ingresará el email con el que se registró en la web, si el correo existe en la base de datos, se le enviará un email con todos sus datos a este correo, también está agregada una protección para evitar que alguien haga spam enviando varios correos si es que conoce el email del usuario.

El sistema se compone de 2 archivos:

[b]ucontrasena.php[/b]
[b]ucontrasenaenviar.php[/b]

Crea el archivo [b]ucontrasena.php[/b] con el siguiente código:

Si te fijas hay un campo oculto que no es visible al usuario, esto se hace ya que al enviar la información de un formulario oprimiendo enter, si es un sólo campo, ignora el botón enviar, y como en el botón enviar hay una variable, hay errores al momento de procesar el formulario, para esto agregamos otro campo más para que el navegador vea 2 campos y de esta forma la información sea enviada correctamente.

[b]Explicación de uso[/b]

Puedes incluír la página [b]ucontrasena.php[/b] en otra página, la página de [b]ucontrasenaenviar.php[/b] debe estar totalmente sóla.
